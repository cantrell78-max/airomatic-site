**Airomatic** is officially on the web at [airomatic.ai](https://airomatic.ai).

This is an AI startup focused on practical tooling — and on showing the work. The first project is this site itself: Astro, dark schematic theme, blog, and a dedicated **Live** page for YouTube sessions.

**What's different from a typical launch:** most of this codebase is being written during live streams with **Grok Build**, xAI's AI coding agent. You can watch architecture, components, and deploy steps happen in real time.

**On the blog:** build logs, session recaps, and embedded video. Set `youtubeId` in any post's frontmatter to embed a stream or VOD.

**Next up:** YouTube channel setup, first live session, Formspree contact form, and Cloudflare deploy. [Get in touch](/contact/) if you want to collaborate or suggest a stream topic.
